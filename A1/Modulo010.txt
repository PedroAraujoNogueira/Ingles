LESSON SEVENTY THREE

to buy - comprar.
bought - buy(no passado). //
gift - lembrancinha/presente.
present - presente.
I cooked dinner. - Eu cozinhei o jantar.
I bought this gift. - Eu comprei esse presente. //
for him - para ele. //
I bought a present for him, not for me. - Eu comprei um presente para ele, não para mim. // 
Pronomes sujeito -> Pronomes objeto //
I -> me //
you -> you //
he -> him //
she -> her //
it -> it //
we -> us //
they -> them //
He cooked dinner for her. - Ele cozinhou o jantar para ela. //
They bought a gift for us. - Eles compraram um presente para nós. //
We said the message is for you. - Nós dissemos que a messagem é para você.
I can't talk to them now. - Eu não posso falar com eles agora.
I cooked dinner for us, not for me. - Eu cozinhei o jantar para nós, não para mim.
This message is for him. - Essa mensagem é para ele. //

LESSON SEVENTY FOUR

Quando estamos falando sobre habilidades que sabemos ou não fazer usamos o verbo can. //
O verbo can não muda de acordo com a pessoa, ele continua sempre o mesmo. //
I can sing. - Eu sei cantar. //
My sister can sing. - Minha irmã sabe cantar.
I can swim. - Eu sei nadar.
I can't cook. - Eu não sei cozinhar.
She can't dance. - Ela não sabe dançar. //
well - bem. //
very well - muito bem. //
They can speak english. - Eles sabem falar inglês. //
I can't drive a bus, but I can drive a car. - Eu não sei dirigir um ônibus, mas eu sei dirigir um carro.

LESSON SEVENTY FIVE

children - crianças. 
I like children. - Eu gosto de crianças.
to teach - ensinar. //
can - usado para falar de coisas que sabemos(ou não sabemos), conseguimos(ou não conseguimos) ou podemos(ou não podemos) fazer. 
I like children, but I can't teach them. - Eu gosto de crianças, mas eu não sei ensiná-las. //
I can make cakes, so I can help.
to say - dizer.
chinese - chinês.
china - china.
I can say Hi in chinese. - Eu sei dizer Oi em chinês.
so - logo/então/consequentemente. //
I can say Hi in chinese, so I can go to China. - Eu sei dizer Oi em chinês, logo eu posso ir para China. //
musical instruments - instrumentos musicais.
I like musical instruments, but I can't play them. - Eu gosto de instrumentos musicais, mas não sei tocá-los.

LESSON SEVENTY SIX

I'm going to - Eu vou. //
OBS1: O "going to" vem sempre acompanhado do verbo TO BE. Então o chunck será sempre "Verbo TO BE" + "going to". //
OBS2: O presente contínuo(gerúndio), "I'm going to", pode ser usado para expressar o futuro em algumas situações, principalmente quando se tem um plano 
ou uma intenção definida. Isso é comum em inglês coloquial. Por exemplo: //
I'm going to the store later. - Eu vou para a loja mais tarde. //
I'm going to the gym tomorrow. - Eu vou para a academia amanhã. //
I'm going to the gym next week. - Eu vou para a academia próxima semana. //
I'm not going to the gym tomorrow. - Eu não vou para a academia amanhã.
I'm going to clean my bedroom tomorrow. - Eu vou limpar meu quarto amanhã.
He's going to the school next week. - Ele vai para a escola semana que vem. //
We're going to go to the beach this weekend. - Nós vamos ir para a praia esse final de semana. //
We're going to the beach this weekend. - Nós vamos para a praia esse final de semana.
OBS3: Para a negativa do "going to" modificamos sempre o verbo TO BE e apenas o verbo TO BE. //
I'm not going to clean my bedroom tomorrow. - Eu não vou limpar meu quarto amanhã. //
He isn't going to school next week. - Ele não vai para escola próxima semana.
We aren't going to the beach this weekend. - Nós não vamos para a praia esse final de semana.
I'm going to travel next year. - Eu vou viajar próximo ano. //
I'm going to live in a different country next year. - Eu vou morar em um país diferente próximo ano.

I'm going to pack for a month to travel to a different country. - Eu vou fazer as malas por um mês para viajar para um país diferente.

LESSON SEVENTY NINE

where - onde.
Where are we going to have dinner tonight? - Onde nós vamos jantar hoje à noite? 
OBS: note que na frase interrogativa o verbo auxiliar vai para o início da frase. EX: //
We are going to have dinner. - Nós vamos jantar. //
Are we going to have dinner. - Nós vamos jantar? //
what - o que/qual.
What are we going to have for dinner? - O que nós vamos comer no jantar?
What are you going to cook for us? - O que você vai cozinhar para nós?
What is he going to cook for us for lunch? - O que ele vai cozinhar para nós no almoço?
